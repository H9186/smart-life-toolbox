# 🚀 Smart Life Toolbox | 生活優化工具箱

歡迎來到我的日常自動化實驗室！這裡收集了我為了提升生活效率、實現財務自由所開發的各種小工具。

## 🛠 目前收錄工具

### 💰 理財分配器 (`/finance`)
* **核心邏輯**：進取型 3322 分配法（30% 穩健投資 / 30% 成長投資 / 20% 儲蓄 / 20% 生活）。
* **適用對象**：處於資產快速累積期、追求複利極大化的實踐者。
* **功能**：自動試算分配額，並提供當月生活費結餘提醒。
